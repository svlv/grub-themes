# GRUB themes for Linux

## Installation

- Copy the themes to `/boot/grub/themes/`

- Edit `/etc/default/grub` file. Find the `GRUB_THEME` variable and change it to `GRUB_THEME=/boot/grub/themes/<THEME_FOLDER>/theme.txt`

- Run `sudo update-grub`
